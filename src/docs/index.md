# USP Network Authentication System©

USP Network Authentication System© (USP NAS) provides a central, scaleable network access control solution for connected devices, 
completely brand agnostic and without the need for agents.

![NAS System Overview](assets/images/nas_screenshot_1.png)

## Basic Feature set

* Network device monitoring via SNMP
* IEEE 802.1x authentication (EAP-TLS) / MAC address authentication (MAB)
* RADIUS Accounting / CoA support
* Integrated RADIUS server / RADIUS proxy
* Dynamic VLAN assignment
* WLAN Support
* ARP table import / DNS zone import
* Inventory import from multiple inventory systems
* Extensive reporting functionality incl. scheduled reporting
* Integrated scripting engine
* Web-based management interface
* REST-API

## Add-on Modules

* Active/passive High-avaliability
