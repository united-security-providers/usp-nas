16.5 - 20 January 2026

Enhancements

  • RADIUS: Added RADIUS enforcement configuration for Alcatel Q-Bridge MIB adaptor (#291977)

Changes

  • Modern GUI: Updated Apache Tika content detection library (used during file uploads) to address a vulnerability reported as critical (CVE–2025–66516) (#291968)

Bugfixes

  • Backup: Fixed an issue preventing large backup files to be restored via GUI (process was aborted due to max. process runtime of one minute). This is a regression introduced in 16.0 (#291964)
  • Modern GUI: Fixed an issue on the LDAP configuration page which did not update LDAP system settings properly. (#291967)
